    Python数据预处理边学边记

    数据千万条，预处第一条；数据不处理，结论两行泪。
    这两句话是本人亲身经历的总结。不论我们在工作中还是研究中，数据的预处理是首位的。通过数据预处理，我们能够更加清晰的认识数据、了解数据。同时，我们还能通过数据预处理保证数据的质量。保证数据的质量就是保证研究和工作的质量。
    1.数据缺失值

    我们在处理数据时最常见情况就是数据缺失值。在python中NaN、NA和None均算作缺失值。我们通常选择删除缺失值和填充缺失值。
    常用删除缺失值函数如下：

    dropna() #删除序列和数据框中数据的缺失值
    from numpy import nan as NA
    data = [1,2,NA,3,4]
    data.dropna()
    data1 = pd.DataFrame([[1,2,3],[NA,2,3]])
    data1.dropna()

    # 数据框中默认dropna()是删除行 axis=1时删除列 
# 传入 how = "all" 时 为删除所有值为NA的行
data2 = pd.DataFrame([[1,2,NA],[2,3,4],[NA,NA,NA]])
data2.dropna(how = "all")

    fillna(value,method,axis,inplace,limit) 

    #使用value来填充缺失值 method参数 ffill目标值向下填充 bfill 目标值向上填充
data2.fillna(0) 
#我们经常平均值和众数填补缺失值
data2.fillna(data2.mean())
data2.fillna(data2.mode())

    -isnull() #表明哪些值是缺失值的布尔值
    -notnull() #isnull()的反函数
    2.数据转换

    在进行数据分析时，我们更多时间是数据转换，将数据转换成为我们需要的形式。
    重复值处理。duplicated drop_duplicated() 原则上保留第一个观测的值
    data = pd.DataFrame({‘k1’:[‘one’,’two’]*3+[‘two’],’k2’:[1,1,2,3,3,4,4]})
    data.duplicated()
    data.drop_duplicates() 

    #基于某列进行去重
data.drop_duplicates(['k1','k2'])

    替代值 data.replace([a,c],b) 将a,c换成b 轴索引
    data.replace([-999,1000],np.nan)
    离散化 cut函数 qcut()函数 基于样本百分数进行分箱 cats = pd.qcut(data,4)
    ages = [20,22,21,25,27,23,37,45,61,45,44]
    bins = [19,25,25,60,100]
    cats = pd.cut(ages,bins)
    cats #返回值为特殊categorical对象

    #统计箱数
pd.values_counts(cats)
cats = pd.qcut(data,4)
