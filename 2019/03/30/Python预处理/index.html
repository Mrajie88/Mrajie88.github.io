    数据千万条，预处第一条；数据不处理，结论两行泪。
    这两句话是本人亲身经历的总结。不论我们在工作中还是研究中，数据的预处理是首位的。通过数据预处理，我们能够更加清晰的认识数据、了解数据。同时，我们还能通过数据预处理保证数据的质量。保证数据的质量就是保证研究和工作的质量。

    -isnull() #表明哪些值是缺失值的布尔值
    -notnull() #isnull()的反函数
